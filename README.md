### Features
* Show all of crypto detail.
* Dark and light theme.
* Buy and sell crypto.
* Sign up, login, logout and edit user account information.
* Show crypto changes per 24h with chart.
* - and you can add anything else. -

### Dev Features
* Flutter framework & Dart language
* Riverpod Provider
* DIO
* flutter lint
* Mock test
